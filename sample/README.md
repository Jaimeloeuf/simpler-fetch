# test
Sample webapp and corresponding API server to demonstrate the use of this library.


## API server
1. Install dependencies
    ```shell
    npm install
    ```
1. Run the API server
    ```shell
    npm run serve
    ```


## webapp
1. Install dependencies
    ```shell
    npm install
    ```
1. Run the webapp
    ```shell
    npm run serve
    ```
1. Navigate to http://localhost:8080
